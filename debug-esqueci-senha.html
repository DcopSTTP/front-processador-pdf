Teste de Redirecionamento - Esqueci a Senha

Passos para testar:

  1. Abra o console do navegador (F12)
  2. Acesse http://localhost:3000/forgot-password
  3. Digite um CPF válido: 123.456.789-10
  4. Clique em "Enviar Código"
  5. Observe os logs no console

O que foi corrigido:

  • Adicionado logs de debug no componente EsqueciSenha
  • Corrigido interceptor do HttpService para não verificar token em rotas públicas
  • Adicionado formulário com preventDefault para evitar submissão indesejada
  • Corrigido tipo dos botões para evitar comportamento padrão
  • Melhorado tratamento de erros no UserService

Possíveis causas do problema:

  • Token expirado causando redirecionamento automático
  • Interceptor do axios redirecionando em caso de erro
  • Formulário sendo submetido sem preventDefault

Como verificar no console:

    1. Procure por: "Iniciando processo de recuperação de senha"
    2. Procure por: "Enviando requisição para esqueci senha"
    3. Procure por: "Token expirado" (não deve aparecer)
    4. Verifique erros de rede na aba Network
